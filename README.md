# docker-deploy
Deployment com Docker (Apresentação Prototipo)

# Clonar Repositorio Adequadamente

```
git clone --recursive https://github.com/Semear-Curiosidade/main
```

# Fazer alterações

Cada submodulo **(server, website)** tem que fazer seus próprios commits, caso necessário, seus próprios branches para mudanças; São repositorios aninhados.

As mudanças nesse repositório só afetam esse repositório e as mudanças nos outros repositórios só afetam aqueles repositórios.
